# EDA

This folder contains a lightweight exploratory data analysis for the credit card fraud dataset.

## Contents

- `eda_analysis.py` - reproducible script that loads `../credit_card.csv`, computes summary statistics, and saves plots.
- `eda_report.md` - written summary of the dataset patterns and business-relevant insights.

## How to run

From the project root:

```bash
python EDA/eda_analysis.py
```

The script prints summary metrics to the console and saves charts into `EDA/plots/`.
